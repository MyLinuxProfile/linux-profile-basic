# Parameters

## URL

- Input:

```
linuxp profile --url https://linuxprofile.com/linux_profile.json
```

## OUTPUT

- Input:

```
linuxp profile --url https://linuxprofile.com/linux_profile.json --output 'my_linux'
```

## SWITCH

- Input:

```
linuxp profile --switch 'my_linux.json'
```

## LIST

- Input:

```
linuxp profile --list
```
