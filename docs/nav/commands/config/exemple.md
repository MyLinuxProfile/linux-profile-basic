# Parameters

## URL

- Input:

```
linuxp config --url https://linuxprofile.com/linux_config.json
```

## TEXT EDITOR

- Input:

```
linuxp config --editor lvim
```

### Options

- vi
- vim
- nano
- lvim
- emacs
- code

## SHELL

- Input:

```
linuxp config --shell bash
```

### Options

- bash
- zsh
- fish
